ImageMagick, FFMpeg, Exiftool, Ghostscript and Dcraw are third party libraries that Razuna needs to work with your assets. You need to install these tools on your server.

The istallation guide for installing these tools are outlined at http://wiki.razuna.com/display/ecp/1.+Install+ImageMagick,+Ghostscript,+FFmpeg+and+Exiftool.

Windows Users:
For your convinience we have included the latest FFMpeg build in this folder.
